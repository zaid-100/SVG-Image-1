The Scale

In chapter 7 of Murray, scales are shown as tools that help us plot co-ordinates. Scales can be given different values and attributes in order to visualize our data as per our needs. One big advantage for using scale is that it makes it easy for us to be flexible with our display, as in, we could use any graduation for our visualization; like a logarithmic scale which is common but even a scale like time for date and time. We can also edit how we want our co-ordinates to appear using scale and we can control the relations of X and Y co-ordinates with respect to the position on the scale.
